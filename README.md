# Helpjuice Realtime Search

<h2>This project permit Users to perform search on articles in realtime</h2>
    - Users register to the app and create articles of their choice;
 
## Built With

- Ruby v2.7.0
- Ruby on Rails v7.0

Ruby: 3.0.0

### Setup

Instal gems with:

```
bundle install
```
```
rails webpacker:install
```
Setup database with:

```
   rails db:create
   rails db:migrate
```
